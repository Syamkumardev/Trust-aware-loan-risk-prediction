# Trust-Aware Machine Learning for Loan Risk Prediction

## Overview

This project develops a trust-aware machine learning framework for loan risk prediction using confidence-based selective automation.

The system predicts whether a customer is likely to default on a loan and uses confidence scores to identify uncertain predictions that may require human review.

## Dataset

Default of Credit Card Clients Dataset (UCI Repository)

- Records: 30,000
- Features: 24+

## Models Used

- Logistic Regression
- Random Forest
- XGBoost

## Techniques

- Selective Prediction
- Confidence Scoring
- Calibration Analysis
- ROC-AUC Evaluation
- Brier Score Assessment
- Hyperparameter Tuning
- 5-Fold Cross Validation
